  • df.head(num) → row 개수가 num에 맞추어서 data 나옴

  • df.info (opens new window)()

    • col 이름들, data 수, data type을 한번에 알 수 있음

  • df.loc[df["col name"].idxmax()] → col에서 가장 큰 값의 나머지 다른 데이터 다 보여줌

    • df.iloc[df["col name"].argmax()] → 같은 정보 나옴

  • sal.groupby("Year").mean()["BasePay"]

    • data를 Year별로 평균 정리한 후 그중 BasePay col만 추출

  • sal["JobTitle"].nunique()

    • JobTitle col에서 유일한 값들의 개수

  • sal["JobTitle"].value_counts()[0:5]

    • JobTitle에서 각 직업별 개수를 나타내고 그 후에 가장 많은 것 5개 불러오기

  • Year == 2013인 것 중에 각 직업별 개수를 나타내고, 그 중에 1개만 있는 것들 개수 세기

    salarr = sal[sal["Year"] == 2013]["JobTitle"].value_counts()
salarr[salarr == 1].count()

sum(sal[sal["Year"] == 2013]["JobTitle"].value_counts() == 1) 도 가능

  • Job Title에서 Chief 단어가 들어간 사람 수 구하기

    # 1. 함수 만들어서 하는 방법
def chief_check(string):
		# ",", "(", ")"를 다 분리해서 string 쪼개고, 소문자로 바꾼 다음, chief 문자 체크
    return "chief" in string.lower().replace(',',' ').replace('(','').replace(')','').split(" ")

# apply로 모든 data에 function 적용하고, sum으로 True 값만 더하기
sum(sal["JobTitle"].apply(lambda x: chief_check(x)))

# 2. 함수 없이 하나 하나 리스트로 만들어서 하는 방법

saljob = sal["JobTitle"].apply(lambda x: x.replace(',',' ').replace('(','').replace(')','').split(" "))
count = 0
for value in saljob:
    for item in value:
        if "chief" == item.lower():
            count += 1
            break
print(count)
