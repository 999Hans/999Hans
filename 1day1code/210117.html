# Plotly and Cufflinks

  • Plotly is an interactive visualization library

    • save, hover(information view), zoom etc

  • Cufflinks connects plotly with pandas

  • 2020 5월 이후부터 참고해야할 것(나올 수 있는 에러 대처법)

    • import chart_studio.plotly as py
    • colorscale='ylorbr'
    • projection={'type':'mercator'}

  • plot 대신에 iplot으로 쓰면 plotly의 기능을 사용할 수 있다

  • df.iplot(kind = "scatter", x = , y = , mode = "markers") → scatter plot, mode를 안 하면 선으로 된 scatter plot 그려짐

    • size로 사이즈 조정 가능

  • df.iplot(kind = "bar", x = , y = )

  • df.sum().iplot() → data에 func을 넣어서 그래프 그릴 수 있음

  • df.iplot(kind = "box")

  • df.iplot(kind = "surface") → 3d surface plot

    • colorscale='ylorbr' 로 색 변화 가능

  • df.iplot(kind = "hist")

  • df.iplot(kind = "spread") → spread plot 그려짐

  • df.iplot(kind = "bubble", x = , y = ) → bubble plot 그려짐(scatter plot에서 marker를 bubble로 변경한 것(원의 크기에 따라 value 차이 보여줌)

  • df.scatter_matrix() → 각 col의 관계를 한번에 보여줌

  • Github page(Cufflinks) (opens new window)에 많은 정보들이 있음

    • 여기에서 cufflinks folder / ta.py (opens new window) 파일에 Technical Analysis에 관련된 내용이 있음
    • 주식 분석, 서로 간의 상관 분석 등 TA에 유용한 tool을 사용하는 방법 있음
